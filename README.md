This repo is a minimal Next.js (pages router) storefront scaffold: product listing, product pages, cart context, and styles.

## Quick start (locally)
1. Install Node 18+.
2. Install deps:
   ```
   npm install
   ```
3. Run dev:
   ```
   npm run dev
   ```
4. Open http://localhost:3000

## Deploy to Vercel
- Push this repo to GitHub, connect to Vercel, and use the default build command `npm run build`.
- Vercel will auto-detect Next.js.
